# 프로젝트 소개

## 서비스 소개

'BankSailor'

"당신만의 금융 항해사, BankSailor와 함께라면 저축이 더 쉬워집니다."

금전을 관리하는 여정은 넓은 바다를 항해하는 것과도 같습니다. BankSailor는 당신만의 안내자로서 여러분의 미래를 설계합니다. 함께 여행하며 안전한 항해를 경험해보세요.
